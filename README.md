## How to install (DEFAULT or BASIC USAGE)
	* git clone https://github.com/kodok154/instatool
	* cd instatools
	* unzip node_modules.zip
	* pkg install nodejs-lts
	* npm instal-instagram-private-api
	* ls
	* Then select the tool you want to use!
	* node filename
<br/>
